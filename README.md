## Instalación

1. Clona este repositorio:

   ```bash
   git clone https://github.com/yhojandev/YonderaTravel.git
   ```

2. Instala las dependencias:

   ```bash
   npm install
   ```

3. Inicia el servidor de desarrollo:

   ```bash
   npm run dev
   ```

4. Visita la página en tu navegador en `http://localhost:4321`.
